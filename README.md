## Telegram-бот для извлечения товаров из чеков с помощью NER-модели и ведения истории покупок.

#### Возможности
- Принимает текст чеков (несколько строк)
- Извлекает названия товаров (NER)
- Сохраняет покупки в SQLite
- Показывает статистику покупок

#### Команды
- `/start` — начать работу
- `/help` — справка
- `/day` — покупки за сутки
- `/week` — покупки за неделю
- `/month` — покупки за месяц

#### Как работает
- Пользователь отправляет строки из чека
- Модель NER извлекает товары
- Товары очищаются и сохраняются в БД

#### Установка
```pip install aiogram transformers torch```

#### Запуск
1. Укажи токен Telegram-бота в .env

	```export BOT_TOKEN="твой_токен"```
  
2. Помести обученную модель в папку model/
3. Запусти:

	```python bot.py```
